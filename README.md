# Stock Market Data Scraper (Python)

A Python script to scrape stock market ticker data (Price, Change, Volume) from a financial news website.

> [!NOTE]
> This project is designed for educational purposes using a mock or public data source. Scraping financial sites like Yahoo Finance or Google Finance is against their ToS for commercial use. This script demonstrates the *technique* of extracting tabular data.

## Features

-   **Multi-ticker Support**: Scrape data for a list of symbols (e.g., AAPL, GOOGL, MSFT).
-   **Data Extraction**: Parses current price, percent change, and volume.
-   **JSON Export**: Saves the snapshot to a JSON file.

## Installation

1.  **Clone the repository**
2.  **Install Dependencies**
    ```bash
    pip install -r requirements.txt
    ```

## Usage

1.  Add tickers to `tickers.txt` (one per line).
2.  Run the scraper:
    ```bash
    python run_scraper.py
    ```
